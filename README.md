# Market Entry Tool

## Description

The **Market Entry Tool** is an application designed to help businesses assess and plan their market entry strategy. It provides resources, guidelines, and essential data required for businesses looking to expand into new markets. The tool offers insights into various market entry models, legal and regulatory information, and potential risks associated with entering different markets.

## Features

- **Product Categories**: Displays a variety of product categories to assist in decision-making.
- **Market Entry Models**: Offers detailed descriptions of different market entry strategies such as direct investment, joint ventures, and more.
- **Legal and Regulatory Resources**: Provides valuable legal resources and regulatory guidelines for businesses expanding into new markets.
- **Risk Assessment**: Helps evaluate potential risks when entering new markets.
- **Integration with External APIs**: Retrieves the latest data on tariffs, duties, and market conditions from external sources.

## Installation

To run this app locally, follow these steps:

1. **Clone the repository**:
   ```bash
   git clone https://github.com/Keerthana-T-P/market-entry-tool.git
   ```

2. **Navigate into the project directory**:
   ```bash
   cd market-entry-tool
   ```

3. **Install dependencies**:
   ```bash
   npm install
   ```

4. **Run the app**:
   ```bash
   npm start
   ```

   The app will run on `http://localhost:3000`.
